W3 Software

This is a list of products related to the WWW project with a link to a status summary of each one.

The common code library is the basis for most browsers. It contains the network access code and format handling.

Clients

  • Line Mode Browser – Not as flashy as a windows implementation, but useful for users without window facilities.
  • Curses Browser – Full-screen interface with character-grid terminal.
  • Macintosh Browser – Pre-alpha version for Mac.
  • Erwise – XMotif browser. (Unsupported)
  • ViolaWWW – Browser for X11.
  • tkWWW – Browser for X11.
  • MidasWWW – Browser for XMotif.
  • Browser-Editor on the NeXT – Editor for NeXTStep with wysiwyg support.
  • Browser on CERNVM – Browser for VM (Nonexistent).
  • Tom Fine’s Browser – TTY-based browser in Perl. (Beta)

Servers

  • VM server – Server for VM mainframes.
  • Generic server – For Unix and VMS systems.
  • VMS Help gateway – Reads VMS/Help files via web link.
  • Mail Robot – Returns documents by mail and manages mailing lists.
